# Hitcounter App
## Interview Assessment

App counts the number of times a site has been visited,
using Redis, and returns this count to the client..

**Writing the Build**
- TASK 1: Python build
-- app.py
- TASK 2: Docker build
-- Dockerfile
- TASK 3: docker-compose
-- docker-compose.yml

**Instrumentation**
- TASK 4: Prometheus
-- prometheus.yml

**Deployment**
- TASK 5: Kubernetes Objects
-- app-deployment.yaml
-- app-ingress.yaml
-- app-service.yaml
-- app-servicemonitor.yaml
- TASK 6: Deployment Script
-- deploy.sh

***Prerequisites***
- Docker
- Minikube
- HELM
- Prometheus
